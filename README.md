# Finance Dashboard

A Streamlit-based financial analysis tool built with Python, Pandas, and YFinance.  
It lets users visualize stock data, moving averages, and trading volume interactively.

## Tech Stack
- Python (Pandas, Matplotlib, Streamlit)
- Yahoo Finance API
- Modular structure for scalability

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
